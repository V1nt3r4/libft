# 📚 **libft** - Your Very Own C Standard Library

Welcome to **libft**, a custom-built implementation of a subset of the standard C library. This is a foundational project for mastering low-level programming in C. In this repository, you will find carefully crafted functions that mirror (and sometimes extend) the original ones from the standard library. Let's dive into the core of how C works!

## 🌟 **Project Overview**

The **libft** project is about creating your own version of essential C standard library functions. This is the cornerstone of becoming a better C programmer and understanding the language inside out. Why reinvent the wheel, you ask? Simple: it’s about learning by doing.

By the end of this project, you will have developed the following skills:

- Deep understanding of pointer manipulation.
- Mastery of memory management.
- Familiarity with the most important standard library functions (and how they work behind the scenes).
